Flask ChartJS

ChartJS and Flask Visualization Demo

ChartJS is a JavaScript library consisted of a collection of chart types that can be integrated into a data vizualization web application. This demo utilizes BootStrap, jQuery, and ChartJS for front-end rendering and Python Flask and MySQL for handling back-end JSON objects and data storage.
